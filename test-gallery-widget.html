How to Get Started

For Business Owners:

  1. Sign up for an account on our platform
  2. Go to Widgets → Image Gallery
  3. Click "Manage Widget" to customize your gallery
  4. Add your own images using the "Add New Image" button
  5. Copy the embed code and paste it into your website
  6. Your custom gallery will appear instantly!

For Website Visitors:

Simply browse the gallery above using the arrow buttons or dots. Each image represents a different style category. The gallery automatically adapts to your device size and supports touch gestures on mobile devices.
